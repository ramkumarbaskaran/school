# School Management


## Installation

1. `npm install`

2. create a `credentials_env.js` file in the root of the repo

   ```javascript
   module.exports = {
       host     : 'localhost',
       user     : '',
       password : '',
       database : ''
   }
   ```

3. Import school.sql file

4. npm start  - to start the server

5. npm test - to run test cases
